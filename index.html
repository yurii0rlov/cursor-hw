Шпаргалка по гіту:

  • git config --global user.name blabla

    Встановлює ваше глобальне ім'я, яке буде видне в коммітах

  • git config --global user.email blabal@bla

    Встановлює глобальний е-мейл, який буде видно в коммітах

  • git init

    Створює в вибраній папці локальний репозиторій

  • git status

    Виводить список файлів в папці з репозиторієм, показує кольором чи відстежується кожен з файлів

  • git add

    Додає файли в комміт, найчастіше після add додають знак . , для додавання усіх файлів

  • git commit

    Вказує що дану версію файлів потрібно зберегти, також через прапорець -m вказуються зміни, що були внесені

  • git log

    Виводить список зроблених комітів, з іменем автора, датою та назвою комміта

  • git checkout

    Змінює робочу гілку на обрану

  • git branch

    Виводить список гілок, та помічає активну. Також створює нову гілку з потрібним іменем яке потрібно написати після git branch *

  • git push

    Відправляє усі внесені комміти в репозиторій

  • git pull

    Отримує зміни з репозиторія

  • git merge

    Зливає усі зміни з вибраної А гілки в активну гілку В

  • git clone

    Копіює репозиторій з лінку, в вибрану папку на комп'ютері.

.gitinore файл

Це файл в якому прописуються файли які потрібно приховати від репозиторія. В основному туди входять конфігураційні чи тимчасові файли та папки

ADVANCED

  • git revert

    Створює новий комміт з відміни змін вибраного комміта, та не видаляє їх з історії

  • git reset

    Видаляє чи відміняє вибраний комміт

  • git rebase

    Для склеювання (злияння) декількох комітів в один

  • git cherry-pick

    Бере зміни з вибраного комміта, та вносить їх в активний комміт

  • git stash

    Дозволяє відкласти внесені зміни, без створення комміта - та внести їх пізніше, у потрібний момент

Підключитись через SSH
